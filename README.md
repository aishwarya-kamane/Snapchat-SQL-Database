The Snapchat Database System is designed to store and manage information related to users, profiles, friendships, messages, and stories. It maintains records of users who communicate on the platform by sending messages, adding friends, and sharing stories.
      The main objective of this project is to understand how SQL tables, relationships, and queries work in a relational database system. It also demonstrates the practical use of SQL concepts for managing and retrieving social media data efficiently.

Usage:Refer to the snapchat_queries.txt file and run the said queries on any suitable software
